# bot_jef

## Installation

- install yarn from https://nodejs.org/en/
- install node from https://yarnpkg.com/en/docs/install
- install ffmpeg from https://www.ffmpeg.org/download.html
- run `yarn`
- setup config file using `config.json.example` as an example

## Running

- Run `yarn run build`
- Run `yarn run main`
